# TanStack + Drizzle + Supabase (Bun)

This project includes a Vite + React client with TanStack Router/Query and a Bun server using Drizzle ORM against a Supabase Postgres database.

## Setup
1. Copy .env.example to .env and replace [YOUR-PASSWORD] with your Supabase database password.
2. Install dependencies:
   - bun install
3. Generate and run migrations (optional for the demo route):
   - bun run db:generate
   - bun run db:migrate

## Run
- Start the API server:
  - bun run dev:server
- Start the client:
  - bun run dev

The client uses a dev proxy for /api to http://localhost:3001.
